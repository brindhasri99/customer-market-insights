# Customer & Market Insights Platform using Data Analytics

## Description
This project analyses e-commerce sales data using Python and exploratory data analysis techniques to generate strategic business insights.

## Key Highlights
- Market-wise and time-based sales analysis
- Product-level revenue insights
- Customer segmentation based on revenue contribution
- Data-backed recommendations for pricing and retention strategies

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
